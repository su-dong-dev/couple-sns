## 🫶🏻 COUPLE-SNS
### 커플들의 추억을 기록하고 추억할 수 있는 커플전용 SNS

### 🙊 사용자
- 여자/남자친구와의 사진을 기록하고 싶은 사람
- 여자/남자자친구와 둘 만의 SNS을 만들고 싶은 사람
- 여자/남자친구가 있는 사람.. 🙈

### ⏰ 개발 기간
- 2023.02 ~ 진행중

### 기술 스택
- Front-end : `JavaScript`, `React-native`
- Back-end : `Java`, `SpringBoot`, `SpringDataJPA`, `SpringSecurity`, `MySQL`, `AWS`, `Junit5`
